# Juan Camilo Escobar — Personal Portfolio

A bilingual (EN/ES) personal portfolio website built with HTML5, CSS3 and vanilla JavaScript. Designed with a dark theme and responsive layout.

---

## Features

- **Bilingual** — full EN/ES language switcher powered by vanilla JS, with `localStorage` persistence and automatic browser language detection
- **Responsive** — three layout states: desktop (>1023px), tablet (≤1023px), mobile (≤767px)
- **Downloadable CV** — language-aware download button serves the correct PDF version based on the active language
- **WhatsApp floating button** — direct contact shortcut fixed to the bottom-right corner
- **Dark theme** — custom design system with CSS custom properties, neon green/cyan accent palette and subtle background gradients
- **Self-contained fonts** — Archivo Black and Open Sans embedded locally, no CDN dependency

## Sections

- **Hero** — Name, role, bio, navigation links and CV download button
- **Skills & Tools** — Web development icons, engineering software icons, language chips
- **Featured Projects** — Project cards with preview image, tags, description and links
- **Contact** — Email addresses, social links (LinkedIn, GitHub, Instagram, X)

---

## Responsive Breakpoints

| Breakpoint | Target                                                          |
| ---------- | --------------------------------------------------------------- |
| > 1023px   | Desktop — base styles apply                                     |
| ≤ 1023px   | Tablet — hero stacks, padding reduced                           |
| ≤ 767px    | Mobile — single column, hamburger nav hidden, separators hidden |
